# 📝 To-Do List App (Flutter)

A simple and functional To-Do List application built using Flutter.  
This app allows users to manage daily tasks efficiently with basic CRUD operations.

---

# 🚀 Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- 🗑 Delete tasks
- ☑️ Mark tasks as completed
- 📱 Clean and responsive UI
- ⚡ Real-time updates using setState()

---

# 🛠️ Tech Stack

- Flutter
- Dart
- Material UI

---

# 📂 Project Structure

lib/
├── main.dart              # App logic & UI
├── model/
│     └── todo.dart        # ToDo data model
└── widgets/
└── todoitem.dart    # Reusable ToDo UI widget

---

# 🧠 Concepts Learned

- Stateful vs Stateless Widgets
- State management using setState()
- Passing data between widgets
- Using callbacks for actions (edit, delete, toggle)
- Working with TextEditingController
- Building reusable components
